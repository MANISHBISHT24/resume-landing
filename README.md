# ResumeFlow — Landing Page

A marketing homepage for **ResumeFlow**, an imaginary product that helps
people build a resume online. Built as a weekend warm-up project for a
full stack internship, using semantic HTML, external CSS, and a small
amount of vanilla JavaScript.

## What this is

A single, static homepage covering:

- Navigation bar
- Hero section
- Stats strip
- Features (6 cards)
- Templates (3 cards)
- Testimonials (3 cards)
- Call-to-action band
- FAQ
- Footer

No backend, no accounts, no form submission or click handlers — those
come in a later assignment.

## Tech

- Semantic HTML5
- CSS (Flexbox + Grid, fully responsive)
- Vanilla JavaScript (`querySelector` + `textContent` only)

## Running locally

Just open `index.html` in a browser, or serve the folder with any
static server, e.g.:

```bash
npx serve .
```
